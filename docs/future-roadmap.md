# 향후 로드맵

## 1단계: 정적 카탈로그

HTML, CSS, Vanilla JavaScript, JSON, CSV로 매체 탐색과 견적 문의를 제공한다.

## 2단계: 데이터 시트 운영

CSV를 기준으로 가격과 매체 정보를 정기 갱신한다. 검수 필요 항목을 줄인다.

## 3단계: CMS 또는 DB 도입

운영자가 직접 수정해야 할 데이터가 안정화되면 CMS 또는 DB를 도입한다.

## 4단계: 관리자 로그인

권한이 필요한 수정 작업이 생길 때 인증을 추가한다.

## 5단계: 실시간 구좌 관리

매체별 예약 가능 여부, 시즌 조건, 단기 집행 가능 여부를 실시간으로 관리한다.

## 6단계: 제안서 PDF 자동 생성

선택한 매체와 예산 조건을 기반으로 제안서 PDF를 자동 생성한다.

## 7단계: 광고 집행 결과 리포트 공유

송출 리포트와 모니터링 결과를 광고주가 확인할 수 있게 공유한다.
