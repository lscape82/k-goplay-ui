# 엘리베이터 광고 샘플 사진

아래 4개 파일을 **첨부해 주신 실제 사진으로 덮어써 주세요** (현재는 임시 대체 이미지).
파일명은 그대로 유지하면 코드 수정 없이 바로 반영됩니다.

- `elevator-apt-1.jpg`   ← 아파트 엘리베이터 (예: 가족이 엘리베이터에서 TV 광고 보는 사진)
- `elevator-apt-2.jpg`   ← 아파트 엘리베이터 TV 근접/현장 사진
- `elevator-office-1.jpg` ← 오피스 로비/엘리베이터 디지털 사이니지 (예: ASEM/KEARNEY 로비)
- `elevator-office-2.jpg` ← 오피스 엘리베이터 홀 (예: Encar 광고 여러 대)

상품별 매핑(assets/js/map.js의 ELEV_IMAGES):
- 타운보드·FMK·G사(아파트) → elevator-apt-*.jpg
- 오피스비즈TV·A사(오피스) → elevator-office-*.jpg
